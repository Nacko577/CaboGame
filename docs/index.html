Tips

  • Online play needs a stable internet connection (Wi‑Fi or mobile data). Play uses a secure relay server WebSocket (wss://).
  • Same Wi‑Fi discovers games on your local network. iOS may ask for Local Network permission—allow it for nearby discovery.
  • If a lobby code doesn’t connect, confirm everyone uses the same transport (online vs same Wi‑Fi) and check spelling of the code.
